# drift_detector

A modular Python project for detecting data drift.

## Installation

To install the project, navigate to the root directory and run:

```bash
pip install .
```

## Usage

Once installed, you can run the command-line tool:

```bash
drift_detector
```

## Development

For development, install in editable mode:

```bash
pip install -e .
```

## Running Tests

```bash
pytest
```
